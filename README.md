# 🎧 Spotify Songs Genre Segmentation

## 📌 Project Description

This project focuses on analyzing Spotify songs data and grouping similar songs using Machine Learning.
By using audio features like energy, danceability, tempo, and loudness, songs are clustered into different groups.

---

## 🎯 Objective

* Perform data preprocessing on Spotify dataset
* Analyze and visualize song features
* Apply clustering to group similar songs
* Build a basic music recommendation idea

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📊 Data Preprocessing

* Removed unnecessary columns (genre, artist name, track name, track id)
* Handled missing values
* Selected only numerical features

---

## 📈 Data Visualization

* Histograms for feature distribution
* Boxplots for outlier detection
* Scatter plots for relationship between features
* Correlation heatmap

---

## 🔥 Model Used

* K-Means Clustering

### Steps:

1. Scaled the data using StandardScaler
2. Used Elbow Method to find optimal clusters
3. Applied K-Means algorithm
4. Assigned cluster labels to songs

---

## 📉 Output

* Songs are grouped into clusters based on similarity
* Visualization shows different clusters using colors

---

## 💡 Conclusion

Songs with similar audio features are grouped together.
This clustering technique helps in building a music recommendation system like Spotify.
